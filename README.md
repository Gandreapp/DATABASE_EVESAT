# DATABASE_EVESAT
Evesat; es un sistema de información diseñado para crear cotizaciones de forma rápida, clara y automática. Los clientes pueden seleccionar los servicios que ofrece la empresa y obtener una cotización, mejorando la experiencia de atención al cliente. Para el Adminitrador (la empresa) aporta beneficios operativos, administrativos y estrategicos que facilitan la gestión de eventos.
